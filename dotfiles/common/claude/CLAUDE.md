# GIT Policy
- Merge only if you're explicitly asked for it.
- Don't put co-autored string at the end of commit message.  
- Keep commits atomic (one logical change)
- Make them self-explanatory
- Split different concerns into separate commits


# General
- **ENGLISH ONLY**: for all code, comments, docs, examples, commits, configs, errors, tests
- **Style**: Prefer self-documenting code over comments
- **Tools**: Use rg not grep, fd not find, tree is installed
